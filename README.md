## Description
This is my personal web cv built to show my new skills using **Vue 3** and **TypeScript**. It presents my skills, projects, education, and contact information in an interactive and responsive single-page application.

This project highlights my frontend development abilities using modern frameworks and type-safe code, providing a professional showcase of my developer profile.

## Tech Stack
- Vue 3
- TypeScript
- HTML5 & CSS3
- Tailwind CSS (optional)
- Responsive design principles

## Features
- Interactive sections: About, Skills, Projects, Education, Contact
- Links to GitHub repositories, LinkedIn, and email
- Responsive design for desktop and mobile
- Built with reusable Vue components and TypeScript for type safety
- Easily deployable to GitHub Pages, Vercel, or Netlify
